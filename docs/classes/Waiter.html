    Class Waiter<T>

    A class that extends a Promise, but with an exposed resolve/reject function. The manager can hold a reference to Waiter reference and resolve based on a decoupled signal. It can pass this instance off as a regular Promise, as to not expose the Resolve/Reject APIs.,
